# Telco Customer Churn Prediction

## Project Overview
Predict whether a customer will churn using Logistic Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn

## Steps Performed
1. Data Cleaning
2. Missing Value Handling
3. Label Encoding
4. Feature Scaling
5. Logistic Regression
6. Model Evaluation

## Results
- Accuracy: 78%
